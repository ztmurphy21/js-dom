jQuery FAQs

What is jQuery?

jQuery is a library of JavaScript functions that you're most likely to need as you develop websites.

Why is jQuery becoming so popular?

Three reasons:

  • It is free
  • It lets you get more done with less time.
  • All of its functions cross-bowser compatible.

Which is harder to learn: jQuery or JavaScript?

For most funcitons, jQuery is significantly easier to learn and use than JavaScript. But remember: jQuery is JavaScript.
